# F1 pit stop prediction
Machine learning project focused on predicting whether a Formula 1 driver will pit on the next lap using race telemetry and strategy-related features. Built a binary classification pipeline with data preprocessing, feature engineering, model evaluation using ROC-AUC, and probability-based predictions for Kaggle competition submissions.

## Problem Statement
Predict whether a Formula 1 driver will pit on the next lap.

## Objective
Build an end-to-end machine learning pipeline covering:
- data ingestion
- EDA
- feature engineering
- model training
- evaluation
- deployment-ready inference

## Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Git/GitHub

## Evaluation Metric
ROC-AUC
